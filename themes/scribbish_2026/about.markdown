### Scribbish 2026

A modern refresh of the classic Scribbish theme, updated for 2026.

#### Features

- **Responsive design** - Works beautifully on all screen sizes
- **CSS Custom Properties** - Easy color and typography customization
- **CSS Grid & Flexbox** - Modern layout system
- **Semantic HTML5** - Proper use of header, main, article, aside, footer
- **Improved typography** - System font stack with fluid sizing
- **Accessible** - Focus states, proper contrast, semantic markup
- **Dark code blocks** - Classic green-on-black terminal aesthetic

#### Colors

Maintains the warm burgundy (#930) accent color from the original Scribbish theme while updating the overall feel to be cleaner and more contemporary.

#### Based on

Original [Scribbish](http://quotedprintable.com/pages/scribbish) by [packagethief](http://quotedprintable.com/)
